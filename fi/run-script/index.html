setFilters(esineiden valikoima, merkkijono)​

Käytetään suodattimien asettamiseen.

Ensimmäinen parametri on joukko esineitä. Jokainen objekti edustaa suodatuksen ehtoa. Se sisältää kolme ominaisuutta:

id : Tämä viittaa tämän sarakkeen tunnisteeseen. Jos tämä sarake on tulos tietokantakyselystä, niin tunnus olisi sarake nimi, jonka olet hakenut kyselystä. Esimerkiksi, jos kysely on SELECT email AS user_email FROM users ja uudelleennimeä email Email taulukon oikeanpuoleisella paneelilla, sarakkeen tunnus olisi edelleen user_email Jos tämä sarake lisätään manuaalisesti käyttämällä oikealla puolella olevassa paneelissa olevaa Lisää sarake -painiketta, voit tarkistaa sen tunnuksen vasemmanpuoleisen paneelin displayedData-osiosta. Tyypillisesti, se on satunnaisesti luotu merkkijono alkaen column, kuten column-320169e8-225a-482d-9853-ad34b3040220.

filterFn : suodatin vastaa tyyppiä. We support the following types: equalTo which is the same as = in SQL, notEqualTo which is the same as <> in SQL, contains which is the same as LIKE '%value%' in SQL, does NotContain which is the same as NOT LIKE '%value%' in SQL, lessThan which is the same as < in SQL, notLessThan which is the same as >= in SQL, notMoreThan which is the same as <= in SQL, empty which is the same as IS NULL in SQL, notEmpty which is the same as IS NOT NULL in SQL, before is used to compare time or date, which is the same as > in SQL, after is used to compare time or date, which is the same as < in SQL.

value : käytetään vertailevan arvon määrittämiseen.

Toinen parametri: kun suodattimen tila objekteja on useita, se määrittää, onko niillä "AND" suhde tai "OR"-suhde. Aseta se merkkijonoksi and "AND"-suhdetta varten ja or "OR"-suhdetta varten.

Tuetut komponentit​

Taulukko

Esimerkki​

Tallenna ILLA Drivelle​

Rakenne​

utils.saveToILLADrive({
fileName:string,
fileData:string,
fileType:string,
kansio:string,
allowAnonymous:boolean,
replace:boolean
})

Kuvaus​

Käytetään tiedoston tallentamiseen ILLA Driveen.

fileName: Tiedoston tunnisteen nimi ILLA Drivelle latauksen jälkeen.

fileData: Base64 tai raaka tiedostotiedot, kuten JSON, CSV, tavallinen teksti jne.

fileType: Yksi seuraavista arvoista: Auto, tavallinen teksti, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Kun käytät "Auto", yritämme määrittää tiedostotyypin, joka perustuu joko tiedostopäätteeseen tai tiedostoon.

folder: Kohdekansio tiedostojen lataamista varten. Jätä tyhjäksi ladataksesi pääkansioon. @ info: whatsthis

allowAnonymous: Käytetään sen määrittämiseen, onko anonyymi käyttöoikeus sallittu. Jos aiot asettaa hakemuksesi julkiseksi, anonyymi pääsy on otettava käyttöön. Jotta tietosi, tiedostot tai kansiot olisivat saatavilla anonyymisti, ne on sijoitettava anonyymiin kansioon.

replace: Käytetään kun samassa hakemistossa on tiedostoja, joilla on sama nimi korvaa olemassa oleva tiedosto uudella tiedostolla tai nimeä uusi tiedosto automaattisesti uudelleen.

Lataa ILLA Drivesta​

Rakenne​

utils.dll adFromILLADrive({
    downloadInfo:[{
        tinyURL:string,
        fileID:string
        }],
    asZip:boolean
})

Kuvaus​

Käytetään ladattaessa tiedostoja ILLA Drivesta.

downloadInfo: joukko esineitä, mukaan lukien tinyURL ja fileID. Voit käyttää komponentin Drive-tiedostonvalitsinta valitaksesi tiedostoja ILLA Drivesta ja saada downloadInfo asemaPicker1.value avulla

asZip: käytetään sen määrittämiseen, ladatatko tiedostot Zip:nä.

Esimerkki​

utils.dll adFromILLADrive({
    downloadInfo:drivePicker1.value,
    asZip:true
})

Lataa​

Rakenne​

utils.dll adFile({
    fileType:string,
    fileName:string,
    data:anyType
})

Kuvaus​

Käytetään viemään tietoja laitteeseesi.

fileType: Yksi seuraavista arvoista: Auto, tavallinen teksti, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Kun käytät "Auto", yritämme määrittää tiedostotyypin, joka perustuu joko tiedostopäätteeseen tai tiedostoon.

fileName: nimi ja laajennus.

data: tiedoston tiedot.

Esimerkki​

//download the data of table as a CSV
utils.dll adFile({
    fileType:"CSV",
    fileName:"text.csv",
    data: taulukko2.dataSourceJS
})
//lataa kuva
utils.d·adFile({
    fileType:"PNG",
    fileName:"text.png",
    tiedot: image1.imageSrc
})

utils.setGlobalDataValue({

Aseta yleiset tiedot​

Rakenne​
utils.setGlobalDataIn({
key:string, 
path:string, 
value:any
})
Kuvaus​

Kun globaali data on array tai objekti, käytä setGlobalDataIn -toimintoa muuttaaksesi tiettyä indeksiä vastaavaa arvoa taulukon muodossa tai muuttaaksesi tiettyä avainta vastaavaa arvoa.

key: globalDatan näyttö.

path: sen arvon indeksi, jota aiot päivittää taulukkoon tai objektin avaimeen. Esimerkiksi, asettaa toinen arvo taulukossa, polku on 1 (indeksi). Määrittääksesi attribuutin nimisen avaimen arvon, polku on avain1.

value: asetettu arvo

Esimerkki​
//objectState is an object: {"key1":"value1", "key2":"value2"}, to set the 
utils.setGlobalDataIn({
key:"objectState", 
path:"key2", 
value:input7.value
})
