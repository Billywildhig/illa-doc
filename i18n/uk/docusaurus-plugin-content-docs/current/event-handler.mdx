## Налаштування конфігурації

### Подія

Використовується для встановлення, як викликати дію. Наприклад, якщо ми хочемо запустити restapi1, коли компонент grid(table) даних оновлюється, ми можемо встановити подію до 'оновлення\`. Якщо ми хочемо запустити restapi2, коли компонент типу даних (табличний) змінюється, ми можемо встановити подію в 'сортування'. Якщо ми хочемо спровокувати рестапія після натискання кнопки, ми можемо встановити подію в 'клік'.

### Дія

#### Дія при переході

Використовується для запуску дії, створеної в цьому додатку

| **параметр**      | **обов'язковий чи не**    | **опис**                               |
| ----------------- | ------------------------- | -------------------------------------- |
| Назва дії         | `обов'язковий`            | Виберіть дію, створену в цьому додатку |

#### Керування компонентами

| **параметр**        | **обов'язковий чи не**    | **опис**                                                                                                                                                   |
| ------------------- | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Компонент           | `обов'язковий`            | Виберіть компонент, доданий на поточну сторінку                                                                                                            |
| Метод               | `обов'язковий`            | Використовується, щоб встановити, що робить компонент керування. Різні методи вимагають налаштування різних параметрів. Деталі пояснень наведено нижче.    |

##### значення набору

Використовується для встановлення значення.

###### Параметр

`Значення`: значення може бути `string` або `number` або `object` або `boolean` або `array`, який залежить від типу компонента.

###### Підтримувані компоненти

| ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Введення тексту, тексту, текстову зону, розділювач, селект, Радіогрупа, Радіо Buttonm, Радіо Buttonm, Дата, час, час, події, Text, Bar, Progress, Circle Progress   | `рядок`                                                           |                                                                                                                                                                         |
| `Привіт світ`                                                                                                                                                       |                                                                   |                                                                                                                                                                         |

| JSON редактор
                                                                                                                                            | `string`
                                                      | `{ "firstName": "Чак", "lastName": "Норріс", "age": 80, "bio": "Кільцевий збір медіафайлів з 1940", "пароль": "ніякий", "telephone": "1-800-KICKASS" }` |

#### Зберегти на диск

##### Параметр

`Ім'я файлу з розширення`: рядок. Назва з розширенням файлу після завантаження на ILLA Drive.

`Дані файлу`: рядок. Base64 або необроблені дані файлів, такі як JSON, CSV, звичайний текст тощо.

`Тип файлу`: рядок. Одне з наступних значень: автоматично, звичайний текст, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). При використанні функції "Авто" ми спробуємо визначити тип файлів на основі розширення файлу або даних про файл.

\`Тека: рядок. Тека призначення для завантаження файлу. Залиште порожнім, щоб завантажити до кореневої теки.

`Дозволити анонімний`: Bolean. Використовується для встановлення, чи дозволено анонімний доступ користувача. Якщо ви плануєте встановити свій застосунок як публічний, має бути увімкнений анонімний доступ. Для безпеки ваших даних, файлів або папок, доступних анонімно, повинні розміщуватися в анонімній теці.

`Замінити`: Boolean. Використовується для установки, коли є файли з таким самим ім'ям в одному каталозі, замініть існуючий файл новим файлом або автоматично перейменуйте новий файл.
