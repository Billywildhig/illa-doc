## Asetusten asetukset

### Tapahtuma

Käytetään sen määrittämiseen, miten toiminto käynnistetään. Esimerkiksi, jos haluamme käynnistää restapi1 kun dataruudukko(taulukko) komponentti on päivitetty, voimme asettaa tapahtuman `virkistää`. Jos haluamme käynnistää restapi2 kun sellainen dataruudukko(taulukko) -komponentti on muutettu, voimme asettaa tapahtuman lajittelemaan muutoksen`. Jos haluamme käynnistää restapi3 kun painiketta napsautetaan, voimme asettaa tapahtuman `klikkaa\`.

### Toiminto

#### Käynnistimen toiminta

Käytetään käynnistämään toiminto, joka on luotu tähän sovellukseen

| **parametri**          | **vaaditaan tai ei**    | **kuvaus**                                        |
| ---------------------- | ----------------------- | ------------------------------------------------- |
| Toiminnon nimi         | `pakollinen`            | Valitse toiminto joka on luotu tähän sovellukseen |

| **parametri**          | **vaaditaan tai ei**    | **kuvaus**                                                                                                                                                                 |

### Aja vain kun

Totuusarvo, jota käytetään määritettäessä, kun tapahtuman käsittelijä voi laukaista. Esimerkiksi, kun input1 on null, tapahtumankäsittelijää ei voitu käynnistää: `{input1.value!==""&input1.value!==undefined}`
