## Opzioni di configurazione

### Evento

Usato per impostare come attivare l'azione. Ad esempio, se vogliamo attivare il restapi1 quando il componente della griglia dati (tabella) viene aggiornato, possiamo impostare l'evento su `refresh`. Se vogliamo attivare il restapi2 quando viene modificato il tipo di componente della griglia (tabella) dei dati, possiamo impostare l'evento in `sort change`. Se vogliamo attivare il restapi3 quando viene fatto clic sul pulsante, possiamo impostare l'evento su `click`.

#### Salva su ILLA Drive

##### Parametro

`Nome file con estensione`: Stringa. Nome con estensione del file dopo il caricamento su ILLA Drive.

`File data`: Stringa. Base64 o dati di file grezzi, come JSON, CSV, testo semplice, ecc.

`Tipo Di File`: Stringa. Uno dei seguenti valori: Auto, testo semplice, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Quando si utilizza "Auto", cercheremo di determinare il tipo di file in base all'estensione del file o ai dati del file.

`Cartella`: Stringa. Cartella di destinazione per il caricamento dei file. Lasciare vuoto per caricare nella cartella radice.

`Permetti anonimo`: Boleano. Usato per impostare se è consentito l'accesso anonimo all'utente. Se si prevede di impostare l'applicazione come pubblico, l'accesso anonimo deve essere abilitato. Per la sicurezza dei dati, dei file o delle cartelle accessibili in modo anonimo devono essere inseriti nella cartella anonima.

`Sostituisci`: Booleano. Usato per impostare quando ci sono file con lo stesso nome nella stessa directory, sostituire il file esistente con un nuovo file o rinominare automaticamente il nuovo file.

### Esegui solo quando

Un valore booleano usato per impostare quando il gestore di eventi potrebbe essere attivato. Ad esempio, quando l'input 1 è nullo, il gestore dell'evento non può essere attivato: `{{input1.value!==""&&input1.value!==undefined}}`
