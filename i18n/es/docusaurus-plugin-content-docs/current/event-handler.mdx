## Opciones de configuración

### Evento

Utilizado para establecer cómo activar la acción. Por ejemplo, si queremos desencadenar restapi1 cuando se actualiza el componente de la tabla de datos, podemos establecer el evento a `refresh`. Si queremos desencadenar restapi2 cuando se cambia el tipo de componente de la tabla de datos, podemos establecer el evento a `sort change`. Si queremos activar restapi3 cuando se hace clic en el botón, podemos establecer el evento a `click`.

Cuando hay varias páginas en la aplicación, esta función se puede utilizar para cambiar entre páginas.

#### Guardar en ILLA Drive

##### Parámetro

`Nombre de archivo con extensión`: String. Nombre con extensión del archivo después de subir a ILLA Drive.

`File data`: String. Base64 o datos de archivos crudos, como JSON, CSV, texto plano, etc.

`File Type`: String. Uno de los siguientes valores: Auto, texto Plano, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Al usar "Auto", intentaremos determinar el tipo de archivo basado en la extensión del archivo o en los datos del archivo.

`Carpeta`: Cadena. Carpeta de destino para subir archivos. Deje en blanco para subir a la carpeta raíz.

`Permitir anónimo`: Bolean. Utilizado para establecer si se permite el acceso anónimo de usuario. Si planea establecer su aplicación como pública, el acceso anónimo debe estar habilitado. Para la seguridad de sus datos, los archivos o carpetas accesibles anónimamente deben ser colocados en la carpeta anónima.

`Reemplazar`: Booleano. Utilizado para establecer cuando hay archivos con el mismo nombre en el mismo directorio, reemplazar el archivo existente con un nuevo archivo o renombrar automáticamente el nuevo archivo.

#### Descargar de ILLA Drive

##### Parámetro

`Descargar información`: una matriz de objetos incluyendo tinyURL y fileID. Puede utilizar el selector de archivos de Drive de componentes para seleccionar archivos de ILLA Drive y obtener la `Información de descarga` a través de unPicker1.value

`Guardar como Zip`: Boolean. Utilizado para establecer si descargar o no los archivos como Zip.

### Sólo ejecutar cuando

Un valor booleano usado para establecer cuando el controlador de eventos podría ser activado. Por ejemplo, cuando la input1 es nula, el manejador de eventos no se pudo activar: `{{input1.value!==""&&input1.value!==undefined}}`
