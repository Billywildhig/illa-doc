## 定義

**データコントローラ** 一般的なデータ保護規制の目的のために。 「データ管理者」とは、個人情報がどのような目的で処理されるかを決定する組織を意味します。 当社の顧客はデータコントローラです。

**データ処理者:** 「データ処理者」とは、データ管理者の個人情報を処理する組織です。 当社はお客様のためのデータプロセッサです。 データ処理者として、当社は一般データ保護規則(以下「GDPR」といいます)の要件に拘束されています。

**データ処理:** データ処理は、個人情報に対して行われる任意の操作または一連の操作(自動化されているかどうか)です。 GDPR のテキストに明示的に列挙されているデータ処理の例: コレクション、記録、整理、構造化、保存、適応、変更、 発信、普及または利用可能にすることによって開示、調整、制限、消去または破壊によって開示、コンサルティング、使用、消去。

**個人情報**個人情報とは、お客様が特定できるお客様に関する情報です。 個人情報には、個人の氏名、住所、電話番号、電子メールアドレスなどの情報が含まれます。 個人情報には、サイトまたは当社のサービスにおける活動に関する情報や人口統計情報など、個人の活動に関する情報も含まれます。 例えば、生年月日、性別、地理的領域、好みなど、この情報のいずれかが、その個人を特定する個人情報とリンクされている場合。 個人情報には、「集計」またはその他の個人を特定できない情報は含まれません。 集計情報は、製品、サービスのグループまたはカテゴリについて収集する情報です。 または個人を特定できないユーザー、または個人識別情報が削除されるユーザー。

## Googleのデータは何に使用されますか?

ILLA クラウドを Google スプレッドシートに接続したい場合 GoogleドライブとGoogleシートからデータを取得するために、Google APIへのアクセスを要求します。 具体的な要求と使用方法は次のとおりです。

1.  OAuthを使用したGoogleスプレッドシートに読み書きモードで接続するには、次の3つのAPIへのアクセスをリクエストします。
    -   `https://www.googleapis.com/auth/drive`
    -   `https://www.googleapis.com/auth/spreadsheets`
    -   `https://www.googleapis.com/auth/drive.file`

ILLA Cloud は、以下の 3 つの API を使用します。

-   ドライブへのアクセス許可を読み取り専用に設定するには、`https://www.googleapis.com/auth/drive`を使用しています。 これにより、ユーザーは、ILLA Cloud を介して Google スプレッドシートにアクセスする際に、ファイルの読み取り、作成、変更、および削除を行うことができます。
-   スプレッドシートにアクセスするには `https://www.googleapis.com/auth/spreadsheets` を使用しています。 これにより、ユーザーはILLA Cloudを介してスプレッドシートデータの読み取り、追加、削除、および変更を行うことができます。
-   ドライブにアクセス可能なファイル名を一覧表示するには、`https://www.googleapis.com/auth/drive.file`を使用します。 これにより、ユーザーは作業したいファイルをすばやく選択できます。

2.  読み取り専用モードで OAuth を使用して Google スプレッドシートに接続するには、次の 2 つの API へのアクセスを要求します。
    -   `https://www.googleapis.com/auth/drive.readonly`
    -   `https://www.googleapis.com/auth/spreadsheets.readonly`

ILLA Cloud は、以下の 3 つの API を使用します。

-   ドライブへのアクセス許可を読み取り専用に設定するには、`https://www.googleapis.com/auth/drive.readonly`を使用しています。 これにより、ユーザーはファイルを読み取ることができ、ILLA クラウドを介して Google スプレッドシートにアクセスする際にファイルを作成、変更、削除することはできません。
-   スプレッドシートにアクセスするには `https://www.googleapis.com/auth/spreadsheets.readonly` を使用しています。 これにより、ユーザーはデータを読み取ることができ、ILLA Cloudを使用してスプレッドシートのデータを追加、削除、変更することはできません。

私たちはあなたのデータを保存しません。データが表示されるたびに、それは直接取得され、APIを通じてあなたに表示されます。 さらに、ILLAはいかなる手段によっても、Googleデータと第三者との接続を共有することはありません。
