## 設定オプション

### イベント

アクションのトリガー方法を設定します。 たとえば、データグリッド(テーブル)コンポーネントが更新されたときに restapi1 をトリガーしたい場合は、イベントを `refresh` に設定できます。 データグリッド(テーブル)コンポーネントが変更されたときにrestapi2をトリガーしたい場合は、イベントを `sort change` に設定します。 ボタンがクリックされたときに restapi3 をトリガーしたい場合は、イベントを `click` に設定します。

#### ILLA ドライブに保存

##### パラメータ

`extension`のファイル名: String. ILLA Driveにアップロードした後のファイル拡張子の名前。

`File data`: String. Base64 または JSON、CSV、プレーンテキストなどの生のファイルデータ。

`File Type`: String. 次のいずれかの値: 自動、プレーンテキスト、JPEG、PNG、SVG、JSON、CSV、TSV、Excel(.xlsx)。 「自動」を使用する場合、ファイル拡張子またはファイルデータのいずれかに基づいてファイルの種類を決定しようとします。

`Folder`: String. ファイルのアップロード先フォルダ。 ルートフォルダにアップロードするには空白のままにしてください。

`anonymousを許可`: Bolean. 匿名のユーザーアクセスが許可されているかどうかを設定するために使用されます。 アプリケーションをパブリックに設定する場合は、匿名アクセスを有効にする必要があります。 データのセキュリティのために、匿名でアクセス可能なファイルまたはフォルダを匿名フォルダに配置する必要があります。

`Replace`: Boolean. 同じディレクトリに同じ名前のファイルがある場合に設定するために使用されます。 既存のファイルを新しいファイルに置き換えるか、自動的に新しいファイルの名前を変更します。

###### パラメータ

`key`: globalData の displayName 。

`path`: 配列またはオブジェクト内のキーで更新する値のインデックス。 例えば、配列に 2 番目の値を設定するには、パスは 1 (インデックス) です。 key1 という名前の属性の値を設定するには、path は key1 です。

`value`: 設定する値
