## Options de configuration

### Evénement

Utilisé pour définir comment déclencher l'action. Par exemple, si nous voulons déclencher restapi1 lorsque le composant de la grille de données est actualisé, nous pouvons définir l'événement à `refresh`. Si nous voulons déclencher restapi2 lorsque le type de composant de grille de données est modifié, nous pouvons définir l'événement sur `sort change`. Si nous voulons déclencher restapi3 lorsque le bouton est cliqué, nous pouvons définir l'événement à `click`.

##### Soumettre

Soumettre les données du formulaire. Lorsque la méthode d'envoi est déclenchée, nous allons d'abord valider les données du formulaire. S'il y a des erreurs de validation, la soumission du formulaire sera évitée, et les messages d'erreur correspondants seront affichés. Ce n'est qu'après la validation réussie que la soumission se poursuit.

#### Enregistrer sur ILLA Drive

##### Paramètre

`Nom du fichier avec extension`: chaîne. Nom avec extension du fichier après avoir téléchargé sur ILLA Drive.

`Données du fichier`: Chaîne de caractères. Base64 ou données de fichiers bruts telles que JSON, CSV, texte brut, etc.

`Type de fichier`: Chaîne. Une des valeurs suivantes : Auto, Texte Plain, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Lorsque vous utilisez "Auto", nous tenterons de déterminer le type de fichier en fonction de l'extension du fichier ou des données du fichier.

`Dossier`: Chaîne. Dossier de destination pour le téléchargement de fichier. Laissez vide pour téléverser dans le dossier racine.

`Autoriser l'anonymat`: Bolean. Utilisé pour définir si l'accès aux utilisateurs anonymes est autorisé. Si vous prévoyez de définir votre application comme public, l'accès anonyme doit être activé. Pour la sécurité de vos données, les fichiers ou dossiers accessibles anonymement doivent être placés dans le dossier anonyme.

`Remplacer`: Boolean. Utilisé pour définir quand il y a des fichiers avec le même nom dans le même répertoire, remplacez le fichier existant par un nouveau fichier ou renommez automatiquement le nouveau fichier.

### Exécuter uniquement lorsque

Une valeur booléenne utilisée pour définir quand le gestionnaire d'événements peut être déclenché. Par exemple, lorsque input1 est null, le gestionnaire d'événements ne peut pas être déclenché : `{{input1.value!==""&&input1.value!==undefined}}`
