## Επιλογές ρυθμίσεων

### Γεγονός

Χρησιμοποιείται για να ορίσετε τον τρόπο ενεργοποίησης της ενέργειας. Για παράδειγμα, αν θέλουμε να ενεργοποιήσουμε το restapi1 όταν ανανεωθεί το στοιχείο πλέγμα δεδομένων (πίνακας), μπορούμε να ορίσουμε το γεγονός σε `ανανέωση`. Αν θέλουμε να ενεργοποιήσουμε το restapi2 όταν αλλάξει το είδος του στοιχείου πλέγματος δεδομένων (πίνακα), μπορούμε να ορίσουμε το γεγονός σε `sort change`. Αν θέλουμε να ενεργοποιήσουμε το restapi3 όταν πατηθεί το κουμπί, μπορούμε να ορίσουμε το γεγονός σε `κλικ`.

### Ενέργεια

#### Ενέργεια ενεργοποίησης

Χρησιμοποιείται για την ενεργοποίηση μιας ενέργειας που δημιουργήθηκε σε αυτήν την εφαρμογή

| **παράμετρος**          | **απαιτείται ή όχι**    | **περιγραφή**                                                 |
| ----------------------- | ----------------------- | ------------------------------------------------------------- |
| Όνομα ενέργειας         | `απαιτείται`            | Επιλέξτε μια ενέργεια που δημιουργήθηκε σε αυτήν την εφαρμογή |

#### Στοιχείο ελέγχου

| **παράμετρος**       | **απαιτείται ή όχι**    | **περιγραφή**                                                                                                                                                                   |
| -------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| Μέθοδος              | `απαιτείται`            | Χρησιμοποιείται για να ορίσει τι κάνει το εξάρτημα ελέγχου. Διαφορετικές μέθοδοι απαιτούν διαφορετικές παραμέτρους για να ρυθμιστεί. Παρακάτω παρέχονται λεπτομερείς εξηγήσεις. |

#### Αποθήκευση στο δίσκο ILLA

##### Παράμετρος

`Όνομα αρχείου με επέκταση`: Συμβολοσειρά. Όνομα με την επέκταση του αρχείου μετά τη μεταφόρτωση στο δίσκο ILLA.

`File data`: Συμβολοσειρά. Base64 ή ακατέργαστα δεδομένα αρχείων, όπως JSON, CSV, απλό κείμενο, κ.λπ.

`Τύπος Αρχείου`: Συμβολοσειρά. Μία από τις ακόλουθες τιμές: Αυτόματο, απλό κείμενο, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Όταν χρησιμοποιείτε το "Auto", θα επιχειρήσουμε να προσδιορίσουμε τον τύπο αρχείου με βάση είτε την επέκταση αρχείου είτε τα δεδομένα αρχείου.

`Folder`: Συμβολοσειρά. Φάκελος προορισμού για μεταφόρτωση αρχείου. Αφήστε το κενό για μεταφόρτωση στο ριζικό φάκελο.

`Επιτρέψτε ανώνυμο`: Bolean. Χρησιμοποιείται για να οριστεί αν επιτρέπεται η ανώνυμη πρόσβαση στο χρήστη. Αν σκοπεύετε να ορίσετε την εφαρμογή σας ως δημόσια, η ανώνυμη πρόσβαση πρέπει να είναι ενεργοποιημένη. Για την ασφάλεια των δεδομένων, αρχείων ή φακέλων που είναι προσβάσιμα ανώνυμα πρέπει να τοποθετούνται στον ανώνυμο φάκελο.

`Αντικατάσταση`: Δυαδικό Χρησιμοποιείται για ορισμό όταν υπάρχουν αρχεία με το ίδιο όνομα στον ίδιο κατάλογο, αντικατάσταση του υπάρχοντος αρχείου με ένα νέο αρχείο ή αυτόματη μετονομασία του νέου αρχείου.
