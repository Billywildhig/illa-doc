## Параметры конфигурации

### Событие

Используется для установки вызова действия. Например, если мы хотим вызвать restapi1, когда обновляется компонент сетки данных, мы можем установить событие на `refresh`. Если мы хотим вызвать restapi2, когда изменится вид компонента сетки данных, мы можем установить событие на «сортировать изменения». Если мы хотим вызвать restapi3 при нажатии кнопки, мы можем установить событие в `click`.

### Действие

#### Действие триггера

Используется для запуска действия, созданного в этом приложении

| **параметр**              | **требуется или нет**  | **описание**                                   |
| ------------------------- | ---------------------- | ---------------------------------------------- |
| Название действия         | «требуется»            | Выберите действие, созданное в этом приложении |

#### Компонент управления

| **параметр**        | **требуется или нет**  | **описание**                                                                                                                                                       |
| ------------------- | ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Компонент           | «требуется»            | Выберите компонент, добавленный на текущую страницу                                                                                                                |
| Метод               | «требуется»            | Используется для установки того, что делает управляющий компонент. Различные методы требуют настройки различных параметров. Подробные разъяснения приводятся ниже. |

#### Сохранить в ILLA Drive

##### Параметр

`Имя файла с расширением`: строка. Имя с расширением файла после загрузки на ILLA Drive.

`Данные файла`: строка. Base64 или необработанные файловые данные, такие как JSON, CSV, простой текст и т.д.

`Тип файла`: строка. Одно из следующих значений: Auto, Plain text, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). При использовании "Auto" мы попытаемся определить тип файла на основе либо расширения файла, либо данных файла.

`Папка`: строка. Целевая папка для загрузки файла. Оставьте пустым для загрузки в корневую папку.

`Разрешить анонимность`: Bolean. Используется для установки разрешения анонимного доступа пользователя. Если вы планируете сделать ваше приложение общедоступным, необходимо включить анонимный доступ. Для безопасности ваших данных, файлов или папок, доступных анонимно, необходимо поместить в папку анонимно.

Заменить\`: логическое значение. Используется для установки, когда в той же директории есть файлы с одинаковым именем, заменить существующий файл новым файлом или автоматически переименовать новый файл.

#### Скачать из ILLA Drive

##### Параметр

`Скачать информацию`: массив объектов, включая tinyURL и fileID. Вы можете использовать компонент Выбор файлов диска, чтобы выбрать файлы из ILLA Drive и получить `Скачать информацию` через DrivePicker1.value

`Сохранить как Zip`: Логический. Используется для установки загрузки файлов как Zip.

#### Скачать

Используется для экспорта данных на ваше устройство.

##### Параметр

`Тип файла`: Одно из следующих значений: Auto, Plain text, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). При использовании "Auto" мы попытаемся определить тип файла на основе либо расширения файла, либо данных файла.

`Имя файла`: имя с расширением.

`Данные`: данные файла.

#### Установить глобальные данные

##### Установка значения глобальных данных

Используется для установки значения gobalData

###### Параметр

`key`: отображаемое Имя globalData.

`value`: значение для задания

##### Установить глобальные данные в

Когда глобальные данные являются массивом или объектом, использовать setGlobalDataIn для изменения значения, соответствующего конкретному индексу в массиве, или для изменения значения, соответствующего конкретному ключу в объекте.

###### Параметр

`key`: отображаемое Имя globalData.

`path`: индекс значения, которое вы собираетесь обновлять в массиве или ключе объекта. Например, для установки второго значения массива путь равен 1 (индекс). Для установки значения атрибута именованного ключ1, путь - ключ1.

`value`: значение для задания
