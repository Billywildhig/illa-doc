# Integritetspolicy

_Last updated: 2023-11-06_ 

\*_Din integritet är viktig för oss. _ Denna integritetspolicy förklarar våra rutiner för insamling, användning och utlämnande av information som vi får via våra Tjänster. Denna sekretesspolicy gäller inte för tredje parts webbplatser, tjänster eller program, även om de är tillgängliga via våra Tjänster.

I denna sekretesspolicy:

1.  Vi hänvisar till vår webbplats som “Webbplatsen”.

2.  Vi hänvisar till alla produkter och tjänster vi erbjuder, individuellt och kollektivt, som ”Tjänster”.

3.  Vi hänvisar till dig, den person eller enhet som besöker vår webbplats eller använder våra tjänster, som "du" eller "din" eller (om du är en köpare av våra Tjänster), vår "kund".

## Definitioner

**Personuppgiftsansvarig:** För allmänna dataskyddsbestämmelser, ”Personuppgiftsansvarig” avser den organisation som bestämmer de syften för vilka och på vilket sätt all personlig information behandlas. Våra kunder är personuppgiftsansvariga.

**Personuppgiftsbiträde:** En ”Personuppgiftsbiträde” är en organisation som behandlar personuppgifter för en Personuppgiftsansvarig. Vi är personuppgiftsbiträdet för våra kunder. Som personuppgiftsbiträde är vi bundna av kraven i dataskyddsförordningen ("GDPR").

**Behandling av personuppgifter:** Behandling av personuppgifter är en operation eller en uppsättning operationer (automatiserade eller inte) som utförs på personuppgifter. Exempel på databehandling som uttryckligen anges i texten i GDPR är: insamling, inspelning, organisering, strukturering, lagring, anpassning, ändring, hämta, konsultera, använda, avslöja genom överföring, sprida eller göra tillgängliga, anpassa eller kombinera, begränsa, radering eller förstörelse.

**Personlig information:** Personlig information är all information som handlar om dig, från vilken du kan identifieras. Personuppgifter omfattar information som namn, adress, telefonnummer eller e-postadress. Personlig information innehåller också information om en persons verksamhet, såsom information om hans eller hennes verksamhet på webbplatsen eller våra tjänster, och demografisk information, såsom födelsedatum, kön, geografiskt område och preferenser, när någon av dessa uppgifter är kopplad till personlig information som identifierar den enskilde. Personlig information innehåller inte "aggregat" eller annan icke-personligt identifierbar information. Aggregat information är information som vi samlar in om en grupp eller kategori av produkter, tjänster, eller användare som inte är personligt identifierbar eller från vilka individuella identiteter tas bort.

## Hur samlar vi in personuppgifter?

I vår tjänst som personuppgiftsbiträde samlar vi in personuppgifter från personuppgiftsansvariga på flera sätt:

1.  Information som du lämnar till oss direkt.

2.  Information som vi kan få från tredje part.

3.  Vi kan få information om dig, inklusive personuppgifter, från andra tredje parter. och kan kombinera denna information med annan personlig information som vi upprätthåller om dig. Om vi gör det styr denna sekretesspolicy all kombinerad information som vi upprätthåller i personligt identifierbart format.

## Vad använder vi Googles data till?

När du vill ansluta ILLA Cloud till Google Kalkylark, vi begär åtkomst till Google API för att hämta data från Google Drive och Google Kalkylark. De specifika förfrågningarna och användningen är följande:

1.  För att ansluta till Google Kalkylark med OAuth i läs- och skrivläge kommer vi att begära åtkomst till följande tre API:
    -   `https://www.googleapis.com/auth/drive`
    -   `https://www.googleapis.com/auth/kalkylblad`
    -   `https://www.googleapis.com/auth/drive.file`

ILLA Cloud använder dessa tre API:er för följande ändamål:

-   Vi använder `https://www.googleapis.com/auth/drive` för att ställa in din enhetsbehörighet till skrivskyddad. Detta gör att du och dina användare kan läsa, skapa, modifiera och radera filer när de går in i Google Kalkylark via ILLA Cloud.
-   Vi använder `https://www.googleapis.com/auth/kalkylblad` för att komma åt dina kalkylblad. Detta gör det möjligt för dig och dina användare att läsa, lägga till, ta bort och ändra åtgärder på kalkylblad genom ILLA Cloud.
-   Vi använder `https://www.googleapis.com/auth/drive.file` för att lista de tillgängliga filnamnen i din Drive. Detta hjälper dig och dina användare att snabbt välja de filer de vill arbeta med.

2.  För att ansluta till Google Kalkylark med OAuth i skrivskyddat läge begär vi åtkomst till följande två API:
    -   `https://www.googleapis.com/auth/drive.readonly`
    -   `https://www.googleapis.com/auth/spreadsheets.readonly`

ILLA Cloud använder dessa tre API:er för följande ändamål:

-   Vi använder `https://www.googleapis.com/auth/drive.readonly` för att ställa in din enhetsbehörighet till skrivskyddad. Detta låter dig och dina användare läsa filer och inget sätt att skapa, modifiera och ta bort filer när de går in i Google Kalkylark via ILLA Cloud.
-   Vi använder `https://www.googleapis.com/auth/spreadsheets.readonly` för att komma åt dina kalkylblad. Detta gör att du och dina användare kan läsa data och inget sätt att lägga till, ta bort och ändra data i kalkylblad genom ILLA Cloud.

Vi lagrar inte dina uppgifter, och varje gång data visas, hämtas de direkt och visas för dig via API:et. Dessutom delar ILLA inte din anslutning till Googles data med någon tredje part på något sätt.

## Vad använder vi dina personuppgifter till?

Vi kommer att använda dina personuppgifter i enlighet med denna sekretesspolicy, för att hjälpa oss att leverera tjänsterna till dig. Någon av de uppgifter vi samlar in från dig kan användas på följande sätt:

1.  För att fungera, underhålla och förse dig med funktioner och funktionalitet för tjänsterna.

2.  Att sammanställa statistik och analyser om användningen av vår webbplats och våra tjänster.

3.  Att anpassa din upplevelse.

4.  För att förbättra vår webbplats och våra tjänster strävar vi ständigt efter att förbättra vårt utbud av webbplatser baserat på den information och feedback vi får från dig.

5.  För att förbättra kundservicen – dina personuppgifter hjälper oss att på ett effektivare sätt svara på dina kundserviceförfrågningar och supportbehov.

6.  För att skicka periodiska e-postmeddelanden — Den e-postadress du tillhandahåller kan användas för att skicka information till dig, meddelanden som du begär om ändringar i våra Tjänster, för att varna dig om uppdateringar, och för att skicka periodiska e-postmeddelanden med information som är relevant för ditt konto.

7.  Om du köper våra Tjänster, så att du kan köpa, förnya och använda en kommersiell licens till våra Tjänster.

8.  Vi kan också använda Personlig information som du tillhandahåller för att skicka e-postmarknadsföring om ILLA produkter och tjänster, bjud in dig att delta i evenemang eller undersökningar eller på annat sätt kommunicera med dig i marknadsföringssyfte. Vi tillåter dig att avanmäla dig från att ta emot marknadskommunikation från oss som beskrivs i avsnittet "Dina val" nedan.

9.  Vi kan behöva din Google OAuth för att läsa dina data i Google Kalkylark när du använder Google Kalkylark.

Vi kan också använda dina personuppgifter när det är nödvändigt för att vi ska kunna uppfylla en rättslig skyldighet, inklusive att dela information med myndigheter och tillsynsmyndigheter när det krävs enligt lag eller som svar på juridisk process, skyldighet eller begäran.

Vi samarbetar med myndigheter och brottsbekämpande myndigheter eller privata parter för att genomdriva och följa lagen. Vi kan komma att lämna ut dina personuppgifter till myndigheter eller brottsbekämpande myndigheter eller privata parter som vi anser vara nödvändiga eller lämpliga: (i) för att svara på krav, juridisk process (inklusive stämning); (ii) att skydda vår egendom, våra rättigheter och vår säkerhet, rättigheter och säkerhet för tredje part eller allmänheten i allmänhet; och (iii) att stoppa all verksamhet som vi anser vara olaglig, oetisk eller rättsligt genomförbar verksamhet.
