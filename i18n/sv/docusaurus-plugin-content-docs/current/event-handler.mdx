## Inställningar för konfiguration

### Händelse

Används för att ställa in hur åtgärden ska utlösas. Till exempel, om vi vill utlösa restapi1 när datagrid(tabell) komponenten är uppdaterad, kan vi ställa in händelsen till `refresh`. Om vi vill utlösa restapi2 när den typ av data grid(tabell) komponenten ändras, kan vi ställa in händelsen till "sortera förändring". Om vi vill utlösa restapi3 när knappen klickas kan vi ställa in händelsen till `click`.

#### Spara till ILLA Drive

##### Parameter

`Filnamn med extension`: String. Namn med tillägg av filen efter uppladdning till ILLA Drive.

`Fildata`: Sträng. Base64 eller rå fildata, som JSON, CSV, oformaterad text, etc.

`Filtyp`: Sträng. Ett av följande värden: Auto, Oformaterad text, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). När vi använder "Auto" kommer vi att försöka bestämma filtypen baserat på antingen filtillägget eller fildata.

`Mapp`: Sträng. Målmapp för filuppladdning. Lämna det tomt för att ladda upp till rotmappen.

`Tillåt anonyma`: Bolean. Används för att ange om anonym användaråtkomst är tillåten. Om du planerar att ställa in din ansökan som offentlig måste anonym åtkomst vara aktiverad. För att dina data ska vara säkra måste filer eller mappar som är tillgängliga anonymt placeras i den anonyma mappen.

`Ersätta`: Boolean. Används för att ange när det finns filer med samma namn i samma katalog, ersätt den befintliga filen med en ny fil eller byt automatiskt namn på den nya filen.

#### Ladda ner från ILLA Drive

`Spara som Zip`: Boolean. Används för att ställa in om du vill ladda ner filerna som en zip.

### Kör endast när

Ett booleskt värde som används för att ställa in när händelsehanteraren kunde utlösas. Till exempel, när input1 är noll, kunde händelsehanteraren inte utlösas: `{{input1.value!==""&input1.value!==odefinierat}}}`
