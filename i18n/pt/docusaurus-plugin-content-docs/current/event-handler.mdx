## Opções de configuração

### Evento

Usado para definir como acionar a ação. Por exemplo, se quisermos acionar restapi1 quando o componente de grade de dados (tabela) for atualizado, podemos definir o evento como `atualizar`. Se quisermos acionar o restapi2 quando o tipo de componente (tabela) de dados for alterado, podemos definir o evento como `classificar mudança`. Se quisermos acionar o restapi3 quando o botão for clicado, podemos definir o evento como `clicar`.

##### Parâmatro

### Executar somente quando

Um valor booleano usado para definir quando o manipulador de eventos pode ser acionado. Por exemplo, quando o input1 for nulo, o manipulador de eventos não pôde ser acionado: `{{input1.value!==""&&input1.value!==undefined}}`
