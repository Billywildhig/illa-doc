setFiltre (array of objects, string)​

Folosit pentru a seta filtre.

Primul parametru este o serie de obiecte. Fiecare obiect reprezintă o condiție de filtrare. Acesta include trei atribute:

id : Acesta se referă la id-ul acestei coloane. Dacă această coloană este rezultatul unei interogări a bazei de date, atunci id-ul ar fi numele coloanei pe care l-ați regăsit din interogare. De exemplu, în cazul în care interogarea este SELECT e-mail AS user_email DE la utilizatori şi redenumiţi e-mail ca Email folosind panoul din partea dreaptă a tabelului, ID-ul coloanei ar fi în continuare user_email dacă această coloană este adăugată manual folosind butonul Add Column din panoul din dreapta, poți să îi verifici id-ul în secțiunea displayedData din panoul din stânga. În mod obișnuit, va fi un șir generat aleatoriu începând cu column cum ar fi column-320169e8-225a-482d-9853-ad34b3040.

filterFn : tipul filtrului se potrivește. Acceptăm următoarele tipuri: equalTo care este la fel ca = în SQL, notEqualTo care este la fel ca <> în SQL, conţine care este la fel ca LIKE '%value%'' în SQL, does NotContain" care este la fel ca NOT LIKE '%value%'' în SQL, lessThancare este la fel ca\<în SQL,notLessThancare este la fel ca>=în SQL,notMoreThancare este la fel ca\<=în SQL,golycare este la fel caIS NULLîn SQL,notEmptycare este la fel caIS NULLîn SQL,beforeeste folosit pentru a compara ora sau data, care este acelaşi cu>în SQL,aftereste folosit pentru a compara ora sau data, care este la fel ca\<` în SQL.

value : folosit pentru a seta valoarea de comparat.

Al doilea parametru: când sunt obiecte cu filtru multiplu, acesta determină dacă au o relaţie "AND" sau o relaţie "OR". Setaţi-l ca şir de caractere pentru o relaţie "AND", şi sau pentru o relaţie "OR".

Componente suportate​

Tabel

Exemplu​
table1.setFilters([{"id":"lună", "filterFn":"equalTo", "value":"May"}, {"id":"lună", "filterFn":"equalTo", "value":"June"}], "sau")

Salvează în unitatea ILLA​

Structura​

utils.saveToILLADrive({
fileName:string,
fileData:string,
fileType:string,
folder:string,
allowAnonymous:boolean,
replace:boolean
})

Descriere​

Folosit pentru a salva un fișier pe ILLA Drive.

fileName: Numele cu extensia fișierului după încărcarea pe unitatea ILLA.

fileData: Base64 sau date de fișier brut, cum ar fi JSON, CSV, text simplu etc.

fileType: Una dintre următoarele valori: Auto, text simplu, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Când utilizăm "Auto", vom încerca să determinăm tipul de fișier bazat fie pe extensia fișierului, fie pe datele fișierului.

folder: Dosar destinație pentru încărcarea fișierului. Lăsați necompletat pentru a fi încărcat în dosarul rădăcină.

allowAnonymous: Folosit pentru a seta dacă este permis accesul anonim al utilizatorului. Dacă intenționați să setați aplicația dvs. publică, accesul anonim trebuie să fie activat. Pentru securitatea datelor dvs., fișierele sau dosarele accesibile anonim trebuie plasate în folderul anonim.

replace: Folosit pentru a seta când există fișiere cu același nume în același director, înlocuiţi fişierul existent cu un fişier nou sau redenumiţi automat fişierul nou.

Exemplu​

: true

key: displayName of globalData aşa cum arată următorul screenshot.

value: valoarea de setat

Exemplu​
/use input5 to get the user input value and set as the value of stringState 
utils.setGlobalDataValue({
key:"stringState", 
value:input5.value
})

Setare date globale în​

Structura​
utils.setGlobalDataIn({
key:string, 
path:string, 
value:any
})
Descriere​

Când datele globale sunt o matrice sau un obiect, utilizaţi setGlobalDataIn pentru a modifica valoarea corespunzătoare unui index specific într-o array sau pentru a modifica valoarea corespunzătoare unei anumite chei dintr-un obiect.

key: displayName of globalData.

cale: indexul valorii pe care intenţionezi să o actualizezi în array sau cheia din obiect. De exemplu, pentru a seta a doua valoare in array, calea este 1 (indexul). Pentru a seta valoarea atributului numit key1, calea este cheia1.

value: valoarea de setat

Exemplu​
/objectState este un obiect: {"key1":"value1", "key2":"value2"}, to set the 
utils.setGlobalDataIn({
key:"objectState", 
path:"key2", 
value:input7.value
})

utils.downloadFromILLADrive({
    downloadInfo:{{drivePicker1.value}},/❌
