setFilters(pole objektů, řetězec)​

Používá se pro nastavení filtrů.

První parametr je pole objektů. Každý objekt představuje podmínku filtrování. Zahrnuje tři atributy:

id : Odkazuje na ID tohoto sloupce. Pokud je tento sloupec výsledkem dotazu v databázi, pak id bude název sloupce, který jste získali z dotazu. Například, pokud je dotaz SELECT email AS user_email FROM uživatelé a přejmenujte email na Email pomocí panelu vpravo v tabulce, id sloupce bude stále user_email Pokud je tento sloupec ručně přidán pomocí tlačítka Přidat sloupec v pravém panelu, můžete zkontrolovat jeho ID v sekci displayedData v levém panelu. Obvykle se jedná o náhodně generovaný řetězec začínající na column jako například column-320169e8-225a-482d-9853-ad34b3040220.

filterFn : typ filtru. Podporujeme následující typy: equalTo což je stejné jako = v SQL, notEqualTo což je stejné jako <> v SQL, contains což je stejné jako LIKE '%value%' v SQL, dělá NotContain, který je stejný jako NELIKE '%value%' v SQL, lessThan, který je stejný jako < v SQL, notLessThan, což je stejné jako >= v SQL, notMoreThan je stejný jako <= v SQL, empty což je stejné jako IS NULL v SQL, notEmpty což je totéž jako IS NULL v SQL, before se používá k porovnání času nebo data, což je shodné s > v SQL, po se používá k porovnání času nebo data, což je stejné jako < v SQL.

hodnota : používá se k nastavení porovnání.

Druhý parametr: pokud existují objekty s více podmínkami, určuje, zda mají vztah "AND" nebo "OR". Nastavte to jako řetězec and pro vztah "AND" a or pro vztah "OR".

Podporované komponenty​

Stupeň úvěrové kvality 1

Příklad​
table1.setFiltery([{"id":"month", "filterFn":"equalTo", "value":"May"}, {"id":"month", "filterFn":"equalTo", "hodnota":"Červen"}], "nebo")

Struktura​

utils.saveToILLADrive({
fileName:string,
fileData:string,
fileType:string,
folder:string,
allowAnonymous:boolean,
replace:boolean
})

L 343, 22.12.2009, s. 1).​

Slouží k uložení souboru na ILLA Drive.

file Name: Název s příponou souboru po nahrání na ILLA Drive.

fileData: Base64 nebo nezpracovaná data souborů, jako je JSON, CSV, prostý text, atd.

fileType: Jedna z následujících hodnot: Auto, prostý text, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Při použití "Auto," se pokusíme určit typ souboru na základě přípony souboru nebo dat souboru.

složka: Cílová složka pro nahrávání souborů. Nechte prázdné pro nahrání do kořenové složky.

allowAnonymous: Používá se k nastavení, zda je povolen anonymní uživatelský přístup. Pokud plánujete nastavit svou aplikaci jako veřejnou, musí být povolen anonymní přístup. Pro zabezpečení vašich dat musí být soubory nebo složky přístupné anonymně umístěny do anonymní složky.

replace: Používá se k nastavení, pokud jsou ve stejném adresáři soubory se stejným názvem, nahradit existující soubor novým souborem nebo automaticky přejmenovat nový soubor.

Příklad​

//use upload1 to select files from device and upload to apps folder. 
utils.saveToILLADrive({
fileName:upload1.files[0].name,
fileData:upload1. Soubory[0].dataURI,
fileType:"PNG",
adresář:"apps",
allowAnonymous:false,
replace:true
})
//nahrajte textový soubor.  
utils.saveToILLADrive({
fileName:"test.txt",
fileData:"abc",
fileType:"prostý text",
folder:"apps",
allowAnonymous:false,
replace:true
})

Stáhnout​

Struktura​

utils.downloadFile({
    fileType:string,
    fileName:string,
    data:anyType
})

L 343, 22.12.2009, s. 1).​

Používá se k exportu dat do vašeho zařízení.

fileType: Jedna z následujících hodnot: Auto, prostý text, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Při použití "Auto," se pokusíme určit typ souboru na základě přípony souboru nebo dat souboru.

název souboru: název s příponou.

data: data souboru.

Příklad​

//download data tabulky jako CSV
utils.downloadFile({
    fileType:"CSV",
    fileName:"text.csv",
    data: table2.dataSourceJS
})
//download obrázek
utils.downloadFile({
    fileType:"PNG",
    fileName:"text.png",
    data: image1.imageSrc
})

utils.setGlobalDataIn({
key:string,

})
L 343, 22.12.2009, s. 1).​

Pokud jsou globální data pole nebo objekt, použít setGlobalDataIn pro změnu hodnoty odpovídající určitému indexu v poli nebo pro změnu hodnoty odpovídající určitému klíči v objektu.

klíč: displayName of globalData.

cesta: index hodnoty, kterou hodláte aktualizovat v poli nebo klíč objektu. Například pro nastavení druhé hodnoty v poli je cesta 1 (index). Pro nastavení hodnoty atributu pojmenovaného klíče je cesta klíče 1.

hodnota: hodnota k nastavení

Příklad​
//objectState is a object: {"key1":"value1", "key2":"value2"}, to set the 
utils.setGlobalDataIn({
key:"objectState", 
path:"key2", 
value:input7.value
})
